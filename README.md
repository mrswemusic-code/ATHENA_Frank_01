# ATHENA

**Adaptive Tactical Heuristic Executive Neural Assistant**

ATHENA is a modular AI Operating System being developed from scratch for Linux.

The objective is not to create another chatbot, but a complete AI Operating System capable of managing hardware, software, services and specialized AI agents.
